# Quantile from weighted data

Quantile from weighted data

## Arguments

- p:

  the quantile probabilities

- names:

  should the result be named

- na.rm:

  remove NAs from observations (and weights)

- x:

  the observations

- w:

  importance weights. Must be some non zero weights, and all must be
  finite. They don't need to sum to one.

- link:

  a link function mapping observations to infinite support
