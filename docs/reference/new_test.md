# Create a new `test` S3 object

Create a new `test` S3 object

## Usage

``` r
new_test(value = NA, .attr = list())
```

## Arguments

- value:

  EXAMPLE: a set of named values to use in the `test` S3 object.

- .attr:

  EXAMPLE: attributes to assign to the `test` S3 object

## Value

a new `test` S3 object
