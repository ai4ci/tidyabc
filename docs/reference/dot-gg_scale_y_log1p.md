# A log1p y scale

A log1p y scale

## Usage

``` r
.gg_scale_y_log1p(..., n = 5, base = 10, sf = 2)
```

## Arguments

- ...:

  Other arguments passed on to `scale_(x|y)_continuous()`

- n:

  the number of major breaks

- base:

  the base for the logarithm

- sf:

  significant figures

## Value

a ggplot scale
